
## To build

`sbt assembly`

## Project Overview

It has

 - Spray (REST API) with it's dependencies (e.g. spray json)
 - Parboiled
 - Scala 2.11 
 - Google Auth stuff
 - Specs2
 - Scalacheck
 - commons-math3 and commons-lang3
 - diffutils
 - scallop
 - scalaz

and

 - fat jar plugin (sbt-assembly)
 - some sbt plugins
